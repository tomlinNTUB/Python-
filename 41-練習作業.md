## 3. 計算牌組分數

### 範例程式
``` python
#-------------------------------------------------
# 函式名稱: check
# 傳入參數: (1)牌組字串
# 回傳值: (1)牌組得分
#-------------------------------------------------
#(自行完成)
    
        

#========================
# 主程式(已完成)
#========================
# 待處理資料
data = ['A32JQ', 'JK288', '88322', '4394K', '349AA', '43JJ3', 'AQKK3', '9439A', 'A922A', 'QJ33J', 'QA243', 'AKK38']

# 逐筆處理
for d in data:
    # 呼叫函式
    m = check(d)

    # 印出結果
    print('牌組:{}, 得分:{}'.format(d, m)) 
```

### 說明
``` python
(1) 一個牌組有5張牌
(2) 牌組中的牌點: '2'得2分, '3'得3分, ..., '9'得9分
(3) 牌組中的牌點: 'A'得10分, 'J'得11分, 'Q'得12分, 'K'得13分
(4) 重覆的牌點只能計算1次, 例如: '2224K'牌組中, '2'出現3次但只能計分1次得到2分, 其他2個'2'不再計分, 總得分 = 2+4+13 = 19
```

### 測試
