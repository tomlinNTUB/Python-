## 1. 計算薪水

``` python
輸入: 
    (1) 職等: 字串, 內容為'A', 'B', 或 'C'
    (2) 加班時數: 整數

輸出: 薪水

處理: 
    (1) 假設職等的輸入值一定是'A', 'B', 'C'其中之一, 不必檢查是否輸入了其他值
    (1) 薪水 = 底薪 + 加班費
    (2) 底薪: A級員工60,000元, B級50,000元, C級35,000元
    (3) 加班費: A級員工每小時300元, B級200元, C級150元
```

``` python
請輸入職等: A
請輸入加班時數: 3
>> 薪水:60,900

請輸入職等: B
請輸入加班時數: 8
>> 薪水:51,600

請輸入職等: C
請輸入加班時數: 10
>> 薪水:36,500
``` 



## 2. 計算派外員工薪水

``` python
輸入: 
    (1) 職等: 字串, 內容為'A', 'B', 或 'C'
    (2) 加班時數: 整數
    (3) 外派國家: 字串, 內容為'中國', '美國', '澳洲'其中之一

輸出: 薪水

處理: 
    (1) 假設職等的輸入值一定是'A', 'B', 'C'其中之一, 不必檢查是否輸入了其他值
    (2) 假設外派國家的輸入值一定是'中國', '美國', '澳洲'其中之一, 不必檢查是否輸入了其他值
    (3) 薪水 = 底薪 + 加班費 + 外派補助
    (4) 底薪: A級員工60,000元, B級50,000元, C級35,000元
    (5) 加班費: A級員工每小時300元, B級200元, C級150元
    (6) 外派補助: '中國'補助12,000元, '美國'補助21,000元, '澳洲'補助18,000元
```

``` python
請輸入職等: A
請輸入加班時數: 5
外派國家: 中國
>> 薪水:73,500

請輸入職等: B
請輸入加班時數: 2
外派國家: 美國
>> 薪水:71,400

請輸入職等: C
請輸入加班時數: 4
外派國家: 澳洲
>> 薪水:53,600

請輸入職等: A
請輸入加班時數: 0
外派國家: 澳洲
>> 薪水:78,000
``` 



## 3. 計算正式員工、計時員工、工讀生薪水

``` python
輸入: 
    (1) 職工類別: 1:正式員工, 2:計時員工, 3:工讀生
    (2) 職等: 字串, 內容為'A', 'B', 或 'C'
    (3) 工作時數: 整數
    
輸出: 薪水

處理: 
    (1) 假設職工類別的輸入值一定是'1', '2', '3'其中之一, 不必檢查是否輸入了其他值
    (2) 假設職等的輸入值一定是'A', 'B', 'C'其中之一, 不必檢查是否輸入了其他值
    (3) 假設輸入的工作時數是整數, 不必檢查是否輸入了錯誤值
    (4) 正式員工的薪水 = 底薪 + 加班費
        (4-1) 底薪: A級正式員工60,000元, B級50,000元, C級35,000元
        (4-2) 加班費: 正式員工的基本工時為155小時, 超過者A級正式員工每小時300元, B級200元, C級150元
                     (假設輸入正式員工的基本工時都至少155小時, 不必處理時數不足的情況)
    (5) 計時員工的薪水 = 每小時時薪 * 工作時數
        (5-1) A級計時員工每小時180元, B級160元, C級140元
    (6) 工讀生的薪水 = 每小時時薪 * 工作時數
        (6-1) 不管那一個職等, 工讀生每小時都是120元         
```
