## 1. 計算薪水

``` python
輸入: 
    (1) 職等: 字串, 內容為A, B, 或 C
    (2) 加班時數: 整數

輸出: 薪水

處理: 
    (1) 假設職等的輸入值一定是A, B, C其中之一, 不必檢查是否輸入了其他值
    (1) 薪水 = 底薪 + 加班費
    (2) 底薪: A級員工60,000元, B級50,000元, C級35,000元
    (3) 加班費: A級員工每小時300元, B級200元, C級150元
```

``` python
請輸入職等: A
請輸入加班時數: 3
>> 薪水:60,900

請輸入職等: B
請輸入加班時數: 8
>> 薪水:51,600

請輸入職等: C
請輸入加班時數: 10
>> 薪水:36,500
```
