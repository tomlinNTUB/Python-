# A11-1 函式-基本使用方式

## (1) 程式範例
``` python
import random

#-------------------
# 自訂函式
#-------------------
def sayHello():
    if random.randint(0, 1)==0:
        print('Hello')
    else:
        print('你好')

#-------------------
# 主程式
#------------------- 
for i in range(10):            
    sayHello()
```
