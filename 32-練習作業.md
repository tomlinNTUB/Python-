# 以comprehension方法產生以下list

## 1.
``` python
[1, 1, 1, ..., 1]

註: 共有1000個元素
```

## 2. 
``` python
[1, 2, 3, ..., 100]
```

## 3. 
``` python
[5, 10, 15, 20, ... , 500]
```

## 4. 
``` python
[2, 4, 6, 8, ... , 1000]
```

## 5. 
``` python
[63, 21, 10, ... , 90]

註: 共有100個1~100的亂數
```

## 6. 
``` python
[30, -23, -29, 3, 45, ... , 2] 

註: 共有100個-50~50的亂數
```

## 7. 
``` python
[5, 7, 10, 14, ..., 1000] 

註: 在1~1000間可被5或7整除的整數
```

## 8. 
``` 
['A', 'B', 'C', ..., 'Z']

註:
chr(65) -> 'A'
chr(90) -> 'Z'
```

## 9. 
``` python
['K', 'Z', 'E', ..., 'E']

註: 產生100個'A'~'Z'之間的亂數
```

## 10. 
``` python
['A1', 'A2', 'A3', 'A4', 'B1', 'B2', 'B3', 'B4', 'C1', 'C2', 'C3', 'C4', ... , 'M1', 'M2', 'M3', 'M4']

註: 
每個'A'~'M'依序搭配'1'~'4'
```
