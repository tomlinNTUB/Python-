## 1. 成績計算

### 範例程式
``` python
#-------------------------------------------------
# 函式名稱: score
# 傳入參數: (1)國文成績, (2)英文成績, (3)數學成績
# 回傳值: (1)總分, (2)平均, (3)不及格科目數
#-------------------------------------------------
#(自行完成)
        

#========================
# 主程式(已完成)
#========================
# 待處理資料
data = [(60, 90, 45), (50, 55, 35), (90, 80, 70), (75, 45, 80), (60, 55, 45)]

# 逐筆處理
for chi, eng, mat in data:
    # 呼叫函式
    tot, avg, fail = score(chi, eng, mat)

    # 印出結果
    print('國文:{}, 英文:{}, 數學:{}, 總分:{}, 平均:{:.1f}, 不及格:{}'.format(chi, eng, mat, tot, avg, fail)) 
```

### 說明
``` python
(1) 總分 = 國文+英文+數學
(2) 平均 = 總分/3
(3) 及格成績為60分
```
