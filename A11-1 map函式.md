# A11-1 map函式

## (1) 程式範例
``` python
#----------------------------------------
# 建立一個清單, 某些元素前後有不必要的空白
#----------------------------------------
a = ['abc  ', 'def ', 'ghi  ', '  jk']

#----------------------------------------
# 將a清單中的每個元素套用str.strip()方法, 
# 去除前後空白, 結果是一個新的可迭代物件
#----------------------------------------
b = map(str.strip, a)

#----------------------------------------
# 印出結果
#----------------------------------------
for k in b:
    print('資料:{}, 長度:{}'.format(k, len(k)))
```


### 測試
``` python
資料:abc, 長度:3
資料:def, 長度:3
資料:ghi, 長度:3
資料:jk, 長度:2
``` 

## (2) 程式範例
``` python
#----------------------------------------
# 建立一個清單, 某些元素前後有不必要的空白
#----------------------------------------
a = ['123', 'abc', '4567', 'abcde'] 

#----------------------------------------
# 將a清單中的每個元素套用str.strip()方法, 
# 去除前後空白, 結果是一個新的可迭代物件
#----------------------------------------
b = map(str.isdigit, a)

#----------------------------------------
# 將迭代物件轉為list
#----------------------------------------
m = list(b)

#----------------------------------------
# 印出結果
#----------------------------------------
print(m)
print('共有數字個數:', m.count(True))  
```


### 測試
``` python
[True, False, True, False]
共有數字個數: 2
```
