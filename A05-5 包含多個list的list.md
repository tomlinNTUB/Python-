# A05-5 包含多個list的list


### (1) 程式範例
``` python
#---------------------------------------------
# list元素內容:
# 序號(0), 性別(1), 國文(2), 英文(3), 數學(4)
#---------------------------------------------

# 一個包括多個list的list
items = [[1,'男',46,56,42], [2,'男',48,51,39], [3,'男',50,52,47], [4,'女',40,56,45],
        [5,'男',43,41,29], [6,'女',44,52,50], [7,'女',44,56,47], [8,'男',50,56,47],
        [9,'男',56,60,30], [10,'男',43,55,49]]

# 逐行印出list的資料
for item in items:
    print(item)
```
