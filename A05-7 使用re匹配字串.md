# A05-7 使用re匹配字串

## (1) 程式範例
``` python
#------------------------------
# 匯入正則表示式模組
#------------------------------
import re

#------------------------------
# 設定[原始字串]及[樣式]
#------------------------------
s = 'A123456789B314567890C220123456DE56789012FGH3456789'
pattern = '[A-Z]'

#------------------------------
# 找出所有匹配字串
#------------------------------
d = re.findall(pattern, s)

#------------------------------
# 印出結果
#------------------------------
print(d)
```

### 樣式說明
| 樣式 | 意義 |
|---------|------|
| [A-Z] | 先是A到Z其中之一 |

### 測試
``` python
['A', 'B', 'C', 'D', 'E', 'F', 'G', 'H']
```

## (5) 程式範例
``` python
#------------------------------
# 匯入正則表示式模組
#------------------------------
import re

#------------------------------
# 設定[原始字串]及[樣式]
#------------------------------
s = 'A123456789B314567890C220123456DE56789012FGH3456789'
pattern = '[A-Z][1-8]+'

#------------------------------
# 找出所有匹配字串
#------------------------------
d = re.findall(pattern, s)

#------------------------------
# 印出結果
#------------------------------
print(d)
```

### 樣式說明
| 樣式 | 意義 |
|---------|------|
| [A-Z][1-8]+ | 先是A到Z其中之一, 續是1到8出現1次或多次 |

### 測試
``` python
['A12345678', 'B3145678', 'C22', 'E5678', 'H345678']
```
