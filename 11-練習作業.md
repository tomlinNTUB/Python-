## 1. IPV4位址的檢查

#### 程式編號: p1101 <p/>

##### 資料來源: https://leetcode.com/problems/defanging-an-ip-address/
```
一個合法的IPV4位址包括4組數字, 每組數字值為0~255之間, 每組數字間有一個小數點, 例如:
0.255.100.200 就是一個合法的IPV4位址.

現在輸入一個字串 判斷是否為合法的IPV4位址?
若是, 顯示"正確";
否則, 顯示"錯誤".
```

#### 測試
``` python
10.13.255.10
>> 正確

256.10.30.200
>> 錯誤

abc.12.34.56
>> 錯誤

100.200.100.200
>> 正確

200.300
>> 錯誤

100.100.100.100.100
>> 錯誤
```
