## 1. 檢查是否完美數

### (1) 定義
```
完美數是一些特殊的自然數, 它所有的真因子（即除了自身以外的因數）的和，恰好等於它本身. 
例如：第一個完全數是6，它有因數1、2、3、6, 除去它本身6外, 其餘3個數相加 1 + 2 + 3 = 6, 恰好等於本身.
```

### (2) 函式功能
``` python
當傳入整數是完美數時回傳True; 否則回傳False.
```
