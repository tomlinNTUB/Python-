## 1. 總分及平均的函式

``` python
# 待處理資料
data = [(70, 85, 90), (80, 87, 90), (77, 50, 60)]

# 逐筆計算總分及平均
for chi, eng, mat in data:
    # 不加權計算
    tot = chi + eng + mat
    avg = tot / 3
    print('總分:{}, 平均:{:.1f}'.format(tot, avg)) 
```

``` python
(1) 增加一個total(chi, eng, mat)函式, 回傳總分
(2) 增加一個average(chi, eng, mat)函式, 回傳平均
```


## 2. (加權)總分及(加權)平均的函式
``` python
# 待處理資料
data = [(70, 85, 90), (80, 87, 90), (77, 50, 60)]

# 逐筆計算(加權)總分及(加權)平均
for chi, eng, mat in data:
    # 不加權計算
    tot = chi + eng + mat
    avg = tot / 3
    print('總分:{}, 平均:{:.1f}'.format(tot, avg)) 

    # 加權計算    
    tot = chi*1.5 + eng*1.0 + mat*2.0
    avg = tot / (1.5+1.0+2.0)
    print('加權總分:{}, 加權平均:{:.1f}\n'.format(tot, avg))   
```

``` python
(1) 增加一個total(chi, eng, mat, wchi=1, weng=1, wmat=1)函式, 回傳加權總分
(2) 增加一個average(chi, eng, mat, wchi=1, weng=1, wmat=1)函式, 回傳加權平均
```
