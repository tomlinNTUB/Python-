``` python
# 主程式
#========================
# 待處理資料(身高, 體重)
person = [(1.7, 50), (1.58, 51), (1.83, 82), (1.65, 63), (1.53, 69)]

# 逐筆計算總分及平均
for h, w in person:
    bmi = value(h, w)
    eva = evaluation(h, w)

    print('BMI:{:.2f}, 評估:{}'.format(bmi, eva)) 
```

### 說明
``` python
(1) bmi = 體重(公斤) / 身高(公尺)的平方
    例如: 某人體重45公斤, 身高159公分, 其bmi = 45 / (1.59*1.59) = 17.799928800284796
    
(2) 衛生福利部公布的成人bmi評估:
    BMI < 18.5 為'過輕',
    18.5 <= BMI < 24 為'正常體重',
    24 <= BMI < 27 為'過重',
    BMI >= 27 即為'肥胖'
    
(3) 完成範例程式的兩個函式    
````
