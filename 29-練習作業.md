## 1. BMI評估

``` python
輸入: 
    (1) 身高(公分)
    (2) 體重(公斤)

輸出: 
    (1) bmi值, 小數2位, 四捨五入
    (2) bmi評語, 過輕, 正常體重, 過重, 肥胖四者其中之一

處理: 
    (1) 輸入的身高及體重均為浮點數
    (2) 體重(公斤) / 身高(公尺)的平方
        例如: 某人體重45公斤, 身高159公分, 其bmi = 45 / (1.59*1.59) = 17.799928800284796
    (3) 衛生福利部公布的成人標準:
        BMI < 18.5 為過輕,
        18.5 <= BMI < 24 為正常體重,
        24 <= BMI < 27 為過重,
        BMI >= 27 即為肥胖
```

請輸入體重(公斤):78

>> 評語:過重
