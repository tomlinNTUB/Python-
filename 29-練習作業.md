## 1. BMI評估

``` python
輸入: 
    (1) 身高(公分)
    (2) 體重(公斤)

輸出: 
    (1) bmi值, 小數2位, 四捨五入
    (2) bmi評語, 過輕, 正常體重, 過重, 肥胖四者其中之一

處理: 
    (1) 輸入的身高及體重均為浮點數
    (2) 體重(公斤) / 身高(公尺)的平方
        例如: 某人體重45公斤, 身高159公分, 其bmi = 45 / (1.59*1.59) = 17.799928800284796
    (3) 衛生福利部公布的成人標準:
        BMI < 18.5 為過輕,
        18.5 <= BMI < 24 為正常體重,
        24 <= BMI < 27 為過重,
        BMI >= 27 即為肥胖
```

``` python
請輸入身高(公分):175
請輸入體重(公斤):78
>> BMI:25.47
>> 評語:過重

請輸入身高(公分):163
請輸入體重(公斤):50
>> BMI:18.82
>> 評語:正常體重

請輸入身高(公分):178
請輸入體重(公斤):51
>> BMI:16.10
>> 評語:過輕

請輸入身高(公分):163
請輸入體重(公斤):77
>> BMI:28.98
>> 評語:肥胖

請輸入身高(公分):156
請輸入體重(公斤):43
>> BMI:17.67
>> 評語:過輕
``` 


## 2. 密碼評估 (練習str可呼叫的方法)

``` python
輸入: 一個字串

輸出: 字串評語, 安全及不安全兩者其中之一

處理: 符合以下項目至少一項者, 視為不安全:
      (1) 長度不足6碼
      (2) 全部是英文, 或全部是數字
      (3) 英文部分都是大寫, 或都是小寫
```

``` python
請輸入檢測字串:abcab
>> 不安全

請輸入檢測字串:123123
>> 不安全

請輸入檢測字串:abc123123
>> 不安全

請輸入檢測字串:abcABC
>> 不安全

請輸入檢測字串:abcABC123
>> 安全

請輸入檢測字串:abAB1
>> 不安全
``` 


## 3. 電腦閱卷

``` python
輸入: 一個學生的答案字串(20個字)

輸出: 與正確答案比對的結果
      20個字, 每個字依序表示該順序的問題是否答對, 若是顯示'.', 若否顯示'*'

處理: 假設標準答案為:ABBDACCDACBCDADAABCD
```

``` python

>> 比義結果: *.*.............*..*

學生答案: ABBDAACDACACDADCABCD

>> 比義結果: .....*....*....*....
``` 
