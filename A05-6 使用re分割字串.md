# A05-6 使用re分割字串


## (1) 程式範例
``` python
#------------------------------
# 匯入正則表示式模組
#------------------------------
import re

#------------------------------
# 設定[待分割字串]及[切割樣式]
#------------------------------
s = 'A10,B5;C3,D4;E10,F15'
pattern = '[,;]'

#------------------------------
# 切斷字串
#------------------------------
d = re.split(pattern, s)

#------------------------------
# 印出結果
#------------------------------
print(d)
```

### 樣式說明
```
[,;] 表示,或;其中之一
```

### 測試
``` python
['A10', 'B5', 'C3', 'D4', 'E10', 'F15']
```
