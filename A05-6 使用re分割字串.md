# A05-6 使用re分割字串


## (1) 程式範例
``` python
#------------------------------
# 匯入[正則表示式]模組
#------------------------------
import re

#------------------------------
# 設定[原始字串]及[樣式]
#------------------------------
s = 'A10,B5;C3,D4;E10,F15'
pattern = '[,;]'

#------------------------------
# 切斷字串
#------------------------------
d = re.split(pattern, s)

#------------------------------
# 印出結果
#------------------------------
print(d)
```

### 樣式說明
| 樣式 | 意義 |
|---------|------|
| [, ; ] | , 或 ; 其中之一 |

### 測試
``` python
['A10', 'B5', 'C3', 'D4', 'E10', 'F15']
```
