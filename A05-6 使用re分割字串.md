# A05-6 使用re分割字串


## (1) 程式範例
``` python
#------------------------------
# 匯入[正則表示式]模組
#------------------------------
import re

#------------------------------
# 設定[原始字串]及[樣式]
#------------------------------
s = 'A10,B5;C3,D4;E10,F15'
pattern = '[,;]'

#------------------------------
# 切斷字串
#------------------------------
d = re.split(pattern, s)

#------------------------------
# 印出結果
#------------------------------
print(d)
```

### 樣式說明
| 樣式 | 意義 |
|---------|------|
| [ , ; ] | , 或 ; 其中之一 |

### 測試
``` python
['A10', 'B5', 'C3', 'D4', 'E10', 'F15']
```


## (2) 程式範例
``` python
#------------------------------
# 匯入正則表示式模組
#------------------------------
import re

#------------------------------
# 設定[原始字串]及[樣式]
#------------------------------
s = 'A123456789B334567890C220123456'
pattern = '[A-Z][12][0-9]{8}'

#------------------------------
# 找出所有匹配字串
#------------------------------
d = re.findall(pattern, s)

#------------------------------
# 印出結果
#------------------------------
print(d)
```

### 樣式說明
| 樣式 | 意義 |
|---------|------|
| [A-Z][12][0-9]{8} | 先是A到Z其中之一, 續是1或2其一, 再是出現8次的0到9 |

### 測試
``` python
['A123456789', 'C220123456']
```
