# A05-3 list的排序及反轉


## (1) list的排序及反轉可用以下方式操作

| 操作敘述 | 意義 |
|---------|------|
| s.sort() | 將序列s的內容由小至大排序 |
| s.sort(reverse=True) | 將序列s的內容由大至小排序 |
| s.reverse() | 反轉序列s的元素 |


## (2) 程式範例
``` python
#----------------------
# 排序list的元素
#----------------------

# 建立一個存放多個成績的list
s = [60, 70, 55, 60, 80, 95]
print('原始值:', s)

# 排序s的內容(由小到大)
s.sort()

# 印出結果
print('排序後:', s)
```

### 測試
``` python
>> 原始值: [60, 70, 55, 60, 80, 95]
>> 排序後: [55, 60, 60, 70, 80, 95]
```


## (3) 程式範例
``` python
#----------------------
# 排序list的元素
#----------------------

# 建立一個存放多個成績的list
s = [60, 70, 55, 60, 80, 95]
print('原始值:', s)

# 排序s的內容(由大到小)
s.sort(reverse=True)

# 印出結果
print('排序後:', s)
```

### 測試
``` python
原始值: [60, 70, 55, 60, 80, 95]
排序後: [95, 80, 70, 60, 60, 55]
```

## (4) 程式範例
``` python
#----------------------
# 反轉list的元素
#----------------------

# 建立一個存放多個成績的list
s = [60, 70, 55, 60, 80, 95]
print('原始值:', s)

# 反轉s的內容
s.reverse()

# 印出結果
print('反轉後:', s)
```

### 測試
``` python
原始值: [60, 70, 55, 60, 80, 95]
反轉後: [95, 80, 60, 55, 70, 60]
```
