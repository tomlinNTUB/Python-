
## 程式範例

#### 請以以下的程式為基礎, 完成各個程式需要完成的格式
``` python
# 輸入字串
s1 = input('請輸入一個字串: ')

# 待比對字串
s2 = "ABCD"

# 逐個檢查字串內容
cnt=0

for s in s1:
    if s in s2:
        cnt+=1

# 印出結果
print('{}共有{}個字元也出現在{}中'.format(s1, cnt, s2))
```

## 執行結果
``` python
請輸入一個字串: ABCdeafaABCd
>> ABCdeafaABCd共有6個字元也出現在ABCD中

請輸入一個字串: abcdabcdab
>> abcdabcdab共有0個字元也出現在ABCD中
```
