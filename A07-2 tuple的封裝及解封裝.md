# A07-2 tuple的封裝及解封裝

#### 不需要特別寫敘述, Python會自動進行tuple的封裝(packing)及解封裝(unpacking)


## (1) tuple的封裝

``` python
#---------------------
# s被封裝成一個tuple
#---------------------
s = 10, 20
print(s)
```

### 測試
``` python
(10, 20)
```


## (2) tuple的解封裝

``` python
#-------------------
# s是一個tuple
#-------------------
s = (10, 20)
print(s)

#-------------------
# s的內容解封裝,
# 分給x及y
#-------------------
x, y = s
print(x)
print(y)
```

### 測試
``` python
(10, 20)
10
20
```


## (3) tuple的封裝及解封裝

``` python
s1 = 10
s2 = 20

#-------------------------------
# 以下可視為(s1, s2) = (s2, s1)
#-------------------------------
s1, s2 = s2, s1

print(s1)
print(s2)
```

### 測試
``` python
20
10
```
