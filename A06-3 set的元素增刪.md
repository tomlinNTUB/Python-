# A06-3 set的元素增刪


## (1) set可用以下方式操作

| 操作敘述 | 意義 |
|---------|------|
| s.add(x) | 將元素x**加入**集合s中** |
| s.remove(x) | **刪去**集合s中的**元素x** |


## (2) 程式範例
``` python
# 建立一個存放多個成績的set
s = {60, 70, 55, 60, 80, 95}
print(s)

# 增加元素
s.add(75)
print(s)

# 刪除元素
s.remove(60)
print(s)
```

### 測試
``` python
>> {70, 80, 55, 60, 95}
>> {70, 75, 80, 55, 60, 95}
>> {70, 75, 80, 55, 95}
```
