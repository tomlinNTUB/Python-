# A12-2 函式-不定個數傳入參數

## (1) 1個或2個傳入參數; 1個回傳值
``` python
#-------------------------------------------
# 計算傭金的函式
# 傳入:銷售金額, 職級(可不傳, 預設為'A')
# 回傳:傭金
#-------------------------------------------
def commission(total, rank='A'):
    if rank=='A':
        if total>20000:
            rate=0.15
        elif total>10000:
            rate=0.1
        else:
            rate=0.08
    else:
        if total>20000:
            rate=0.12
        elif total>10000:
            rate=0.09
        else:
            rate=0.05        
        
    return round(total * rate)


#-------------------------------------------
# 主程式
#-------------------------------------------
t = 25000
r = 'A'

print('銷售{:,}元, 職級{}, 傭金{:,}元'.format(t, r, commission(t, r))) 
```


### 測試
``` python
銷售25,000元, 職級A, 傭金3,750元
``` 

## (2) 3~6個傳入參數; 1個回傳值
``` python
#----------------------------------------------------------
# 一個計算平均的函式
# 傳入: 國文, 英文, 數學,
#       國文加權(預設值1), 英文加權(預設值1), 數學加權(預設值1)
# 回傳: 平均分數 
#----------------------------------------------------------
def average(chi, eng, mat, wchi=1.0, weng=1.0, wmat=1.0):
    return (chi*wchi + eng*weng + mat*wmat) / (wchi+weng+wmat)

#----------------
# 主程式
#----------------
chi=80
eng=65
mat=72

print('國文={}, 英文={}, 數學={}, 平均={:.2f}'.format(chi, eng, mat, average(chi, eng, mat))) 
print('國文={}, 英文={}, 數學={}, 平均={:.2f}'.format(chi, eng, mat, average(chi, eng, mat, 2, 1, 1.5)))
```


### 測試
``` python
國文=80, 英文=65, 數學=72, 平均=72.33
國文=80, 英文=65, 數學=72, 平均=74.00
```
