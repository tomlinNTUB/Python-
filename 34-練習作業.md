``` python
#-------------------------------------------------

# 傳入參數: (1)大人人數, (2)小孩人數, (3)幸運字母
# 回傳值: (1)折扣後總金額, (2)折扣金額
#-------------------------------------------------

# 主程式(已完成)

# 待處理資料
data = [(2, 3, 'P'), (2, 1, 'Q'), (2, 4, 'R'), (1, 2, 'J')]

# 逐筆計算折扣後金額及折扣
for adult, children, lucky in data:
    # 呼叫函式
    total, discount = ticket(adult, children, lucky)

    # 印出結果

```

### 說明
``` python
有一個遊樂園每位入場大人收費850元, 每位入場小孩收400元.
這個遊樂園入園時一家人可抽1個幸運字母, 如果抽中
ABKQGTYPWMNYZ其中之一, 折扣1000元
其他沒有折扣.
```
