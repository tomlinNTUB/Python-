請以以下的程式為基礎, 完成各個程式需要完成的格式

## 程式範例

``` python
# 輸入值
a = input('請輸入一個浮點數:')

# 轉成整數
try:
    a = float(a)
except:
    print('請輸入整數!')
    exit()

# 印出格式化前後之值(小數2位, 四括五入)
print('格式前:{}\t格式後:{:.2f}'.format(a, a))
```

## 執行結果
``` python
請輸入一個浮點數:123.456
>> 格式前:123.456  格式後:123.46

請輸入一個浮點數:1234.123
>> 格式前:1234.123 格式後:1234.12
```
