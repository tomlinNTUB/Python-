

## 1. 計算電費

| 付費方式  | 每度(元)|
|:-----------:|:-----------:|
| 120度(含)以下 |  2.10 |
| 121(含)以上   | 3.02  |


#### 說明
``` python
以上電費分2級, 
例如使用電力90度, 每度2.10元; 
如使用電力150度, 每度3.02元.

(使用90度)
電費 = 2.10*90

(使用150度)
電費 = 3.02*150

小數四捨五入.
``` 

#### 測試
``` python
輸入用電度數: 90
>> 用電90度, 電費共:189元

輸入用電度數: 190
>> 用電190度, 電費共:574元
```
